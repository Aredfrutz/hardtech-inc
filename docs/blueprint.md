# **App Name**: HardTech Academy

## Core Features:

- Homepage Content Display: Dynamically display latest announcements, news articles, and highlighted events, providing an engaging and informative landing experience for visitors.
- Course Catalog & Details: Allow users to browse and view detailed information about available training programs, including course descriptions, outlines, and prerequisites.
- Student Enrollment System: Enable prospective students to select courses and submit their enrollment applications directly through the website.
- Responsive UI & Navigation: Ensure the application provides an optimal viewing and interaction experience across various devices and screen sizes, with intuitive navigation menus.
- AI-powered Course Description Tool: An internal tool for administrators to generate comprehensive draft course descriptions, outlines, and key learning outcomes based on minimal keyword inputs, using a generative AI model.
- Data Storage for Content & Enrollments: Securely store course information, announcements, events, and student enrollment data using Firebase Firestore.

## Style Guidelines:

- The overall scheme will be dark, evoking a sophisticated, modern, and tech-centric atmosphere. The primary brand color, a vibrant lime green (#7BF425), will be used for interactive elements like buttons and key highlights, reflecting energy and innovation. The background will be a very dark, subtly green-tinted grey (#191E15), ensuring good contrast and readability for white or light-colored text. An accent color of muted golden yellow (#C3B622) will provide warmth and additional hierarchy for secondary call-to-actions or decorative elements.
- The application will utilize 'Inter', a grotesque-style sans-serif font. Its modern, machined, and neutral aesthetic aligns perfectly with a clean, minimalist tech training center, providing excellent readability for both headlines and body text.
- Clean, modern, and distinct vector-based icons will be used to represent various tools, technologies, and learning concepts. Icons will reinforce the high-tech and educational theme, ensuring immediate visual comprehension without clutter.
- A responsive, grid-based layout will be implemented using Tailwind CSS, ensuring optimal presentation and usability on all device types, from mobile to desktop. Ample whitespace will contribute to a minimalist and focused user experience, with clear content segmentation for easy scanning of information.
- Subtle and functional animations will enhance user interaction, including smooth transitions for page loads, hover effects on buttons and links, and discreet loading indicators. These animations will feel quick and intuitive, without distracting from the core content.
